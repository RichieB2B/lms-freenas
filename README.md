# lms-freenas
Logitech Media Server (aka Squeezebox Server) on FreeNAS

Build LMS software for FreeNAS in a build-jail using the [jail/build-lms.sh](jail/build-lms.sh) script.

See [this thread](http://forums.slimdevices.com/showthread.php?104122-How-to-Installing-LMS-7-9-nightly-in-a-FreeBSD-jail-on-FreeNAS-9-3&p=875193&viewfull=1#post875193) for the full instructions.
